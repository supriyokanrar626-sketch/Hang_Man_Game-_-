🎮 Hangman Game

An interactive and fun Hangman word guessing game built with simplicity and user engagement in mind. Test your vocabulary skills and try to guess the hidden word before you run out of chances!

🧠 How the Game Works
The game randomly selects a hidden word.
You guess the word one letter at a time.
Each correct guess reveals the letter in its correct position.
Each incorrect guess brings you closer to losing the game 😬
You win by guessing the word before the hangman is fully drawn!
🎯 Features

✨ Interactive gameplay
🎲 Random word selection
🔤 Real-time letter reveal
❌ Tracks incorrect guesses
🖼️ Visual hangman progression (ASCII/UI based)
🔁 Replay option for continuous fun

🕹️ Game Flow
Start the game
A blank word (e.g., _ _ _ _) appears
Enter a letter
Game updates instantly:
✅ Correct → Letter fills in
❌ Wrong → Hangman progresses
Repeat until:
🎉 You win (word guessed), or
💀 You lose (hangman complete)
